3D printer

May new 3D printer (Creality Ender 3 V2) arrived today!

I set up Octoprint on my rapsberry pi and tried to print the example cat file.
The first print aborted 5 seconds in.
Tried everything!
It’s something to do with the cables going into the hot end.

Eventually I was able to print a Benchy.
It looks pretty good!
At this stage, benchmarks need to be more difficult.
