3D Printer

My new 3D printer (Creality Ender 3 V2) arrived today! It only took 5 days to arrive, from CZ. This weekend is a long one due to the bank holiday - time to build!

Construction

I set it up very carfeully following YouTube videos. It was difficult to tell which way I had to tighten the concentric casters, but everything went smoothly.

First print

I decided to print a file from the SD card, and timelapse this historic moment. I had already set up Octoprint on my Rapsberry Pi, and so I tried to print the example cat file. 5 seconds in, it aborted. Why??

Octoprint noticed that the connection was lost. I tried it a few times and got the same results. It always fails in exactly the same places. I tried everything! I was eventually somehow able to print a Benchy, but how can I stop this from happening again?

It took 2 hours. Yes, I watched it almost the entire time! (Partly because it was 2am and I needed it to finish so I could sleep)
