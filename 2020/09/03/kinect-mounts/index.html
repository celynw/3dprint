Kinect Mounts

The Kinect V1 needs to be mounted onto the Turtlebot. However, we don’t seem to have copies of the official mounting kit. Instead, there are some printed brackets, but they are splitting on the layers and I think could do with being replaced. Unfortunately, I think it’s designed to house a nut, and we have bolts, so I’ll have to cut a gap for it to enter. (Yes I could model this instead but I’m busy).

Trying with a Z-offset this time. 0.2mm was too high, when using the ‘super quality’ 0.12mm layer height, but 0.1mm was fine. I have upped the temperature back to 60°, but reducing it to 50° after the first layer.

This time, no sign of elephant feet! I was a bit concerned with the density of the brim, but otherwise, very clean (once I trimmed the stringy bits).

Of course, just like the old ones, they don’t fit properly and it’s very difficult to get both screws in. I’ll have to design my own which actually fit.

Revisit

So the first layer is supposed to be squashed down, so it adheres to the bed properly and so the lines join together. I think I might continue to use th 0.1mm Z offset, but instead put down a thicker first layer.

Also, the layer orientation does matter; these broke apart a bit quickly, I’ll print them horizontally with supports next time.
